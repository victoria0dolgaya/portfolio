Mondrian Painting

(Click on the picture to view)

I recreated a Piet Mondrian-style composition using pure HTML and CSS - no images. The layout is built with CSS Grid: fixed track sizes, grid-column/grid-row spans for each block, and a global grid gap with a black background to form the bold lines. Color panels (red, blue, yellow, and neutrals) are placed precisely to match the original proportions.
This project highlights my CSS Grid layout skills, attention to spacing and alignment, and clean, framework-free markup.
